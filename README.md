O programa objetiva calcular a nota de VF necessária depois de inseridas as notas de VE e VC. 
O programa foi feito em linguagem C e grava os valores de entrada das notas de VE e VC, de forma que se alguém colocar um valor inválido o programa é finalizado.
Se a nota necessária para a VF for menor ou igual a quatro a resposta será 4 e ser for maior que quatro o programa dará o valor.
